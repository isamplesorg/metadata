Class: MeasuredValue

simple object to represent measured values with units and uncertainty; could generalize uncertainty to something like 'measure quality'. The 'value' can be a single value, or a preferred value if a range (minValue, maxValue) is reported.

URI: http://resource.isamples.org/0.5/schema/MeasuredValue
