Class: PhysicalSampleRecord¶

This is a data object that is a digital representation of a physical sample. It provides descriptive properties for any iSamples physical sample. IGSN is recommended. Must be a URI that can be dereferenced on the web.

URI: http://resource.isamples.org/0.5/schema/PhysicalSampleRecord
